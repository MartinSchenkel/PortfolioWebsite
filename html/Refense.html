What was the project?

This project was what I wanted to make (but did not mange to) for my intake assignment to get into Breda University of Applied Sciences. It's a 2D Action Game, in which you can only defeat the enemies by reflecting their attacks back at them. This requires a big amount of hand-eye coordination and I can confidently say that I am bad at my own game.

My goal for this project was to publish it to Itch.io at the end. This was to force me to actually finish a project for once instead of abandoning it before the polishing phase.

The game is open-source and you can find the source code Here on my Github.

Particles

For this game, I wanted to be able to create cool VFX, as I think those can really get your game onto another level.

The 2D Sprite Particle System that I created was inspired by Unreal Engines Cascade System. Each Particle System consists of any number of Emitters, which spawn and move the particles themselves. This allows for some really cool VFX.

Menu

I also needed to add a good menu, that allows you to change settings, such as Audio and Keybinds, as well as look at the credits and start/quit the game. For that I created multiple UI Elements (Buttons, Sliders, Scrolling Text etc.) that I reused throughout the different menus.

I was also able to reuse the Settings screen in the pause menu and it works really well. I learned how to get an AppData Path to save the settings in, which is how most games seem to do it.

AI

For the AI, I went simple. Basically what they do is they move to a random X-Coordinate, jumping in semi-random intervals and shooting a projectile in semi-random intervals. This turned out to be fun and challenging enough and showed me that a Fun AI does not have to be a Smart AI.
