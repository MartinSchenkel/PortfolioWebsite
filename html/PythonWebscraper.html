What was the project?

This was just a fun side project for me. My and my friends were looking into creating a website for our "clan". For this, we wanted to display stats of our players in different games, so I decided to create a python webscraper to fetch those stats.

You can look at the scraper source code on my Github.

You can see a WIP Display of the Stats (fetched by my scraper) on the image above.

What did I learn from the project?

This was the biggest Python project I have ever done and I learned a lot about the Syntax and how to work with python.

I made the program write the stats to Json and/or CSV so I learned how to write to those file formats properly.

I learned what an "XPath" is and how to use it to find HTML Elements on a website using selenium. I learned how to write the XPath myself to keep it as short and as consistent as possible.
